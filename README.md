# 任工记工 - 工地记工管理系统

## 项目概述

任工记工是一款专为工地管理设计的记工系统，帮助工头和工人轻松记录和管理考勤数据，计算工资，提高工地管理效率。

### 主要功能

- **考勤管理**：记录工人出勤、缺勤、请假等状态
- **项目管理**：创建和管理多个工地项目
- **工资计算**：自动计算工人工资，支持全职和点工模式
- **数据统计**：生成工资报表和出勤统计
- **电话簿**：管理工人联系信息
- **节日管理**：记录节假日信息
- **报销管理**：记录报销事项，上传凭证图片，支持导出
- **响应式设计**：支持桌面端和移动设备

## 技术架构

### 前端技术栈
- **HTML5**：页面结构
- **CSS3**：样式设计，使用CSS变量和响应式布局
- **JavaScript**：功能实现，模块化设计
- **LocalStorage/IndexedDB**：本地数据存储

### 核心模块
- **EventBus**：事件总线，实现模块间通信
- **StorageManager**：存储管理，处理数据持久化
- **ThemeManager**：主题管理，支持浅色/深色主题
- **UI组件**：对话框、提示等通用组件

## 快速开始

### 1. 环境要求
- PHP 7.0+（用于服务器端支持）
- 现代浏览器（Chrome, Firefox, Safari, Edge）

### 2. 安装与运行

1. **克隆项目**
   ```bash
   git clone <repository-url> d:\xampp\htdocs\jg
   ```

2. **启动服务器**
   - 使用XAMPP或其他PHP服务器
   - 确保服务器运行在 `http://localhost:80` 或其他端口

3. **访问系统**
   在浏览器中打开：`http://localhost/jg/index.php`

### 3. 首次使用

1. **登录**：系统会自动创建默认用户
2. **创建项目**：点击"新建项目"按钮创建工地项目
3. **设置个人信息**：点击"编辑信息"完善个人信息和工资设置
4. **开始考勤**：在日历中点击日期进行考勤记录

## 使用指南

### 考勤管理

1. **记录出勤**：在日历中点击日期，选择"满勤"、"半天"或"缺勤"
2. **设置缺勤原因**：选择"缺勤"后，可设置请假、放假或休息
3. **记录加班**：点击日期后，可添加加班小时数

### 项目管理

1. **创建项目**：点击"新建项目"按钮，填写项目名称和地址
2. **切换项目**：点击项目卡片切换不同工地项目
3. **管理项目**：编辑项目信息，设置项目状态

### 工资计算

1. **设置工资标准**：在个人信息中设置日工资或月工资
2. **选择计算方式**：全职可选择按自然日、法定工作日或出勤天数计算
3. **查看工资统计**：系统自动计算工作天数、加班费用和总工资

### 报销管理

1. **添加报销**：点击"添加报销"按钮，填写标题、金额、日期和备注
2. **上传凭证**：支持上传图片凭证（最大2MB）
3. **查看大图**：点击图片缩略图可全屏查看，支持滚轮缩放和拖动
4. **编辑删除**：每条记录都有编辑和删除按钮
5. **数据统计**：自动计算报销总额、本月报销和报销笔数
6. **导出数据**：支持导出CSV和HTML格式，包含完整图片链接
7. **自动清理**：超过三个月的报销记录自动删除

### 数据管理

1. **导出数据**：导出考勤和工资数据为Excel或PDF
2. **导入数据**：从Excel导入考勤记录
3. **数据备份**：定期备份数据到本地

## 响应式设计

系统采用响应式设计，适配不同设备：

- **桌面端**：完整功能，多列布局
- **平板端**：自适应布局，优化操作体验
- **移动端**：单列布局，触摸友好的界面

## 开发规范

### 1. 命名规范
- 类名：PascalCase（如 `StorageManager`）
- 函数名：camelCase（如 `loadData`）
- 常量：UPPER_SNAKE_CASE（如 `THEME_LIGHT`）
- 私有属性：_prefix（如 `_privateMethod`）

### 2. 代码风格
- 使用ES6+语法
- 模块化设计，避免全局变量
- 适当的注释和文档
- 错误处理和日志记录

### 3. CSS规范
- 使用CSS变量定义颜色和间距
- 采用BEM命名规范
- 响应式设计，使用媒体查询
- 避免内联样式

### 常见问题

**Q: 移动设备端对话框显示异常？**
A: 已修复移动设备端对话框向右偏移的问题，确保在所有设备上正确居中显示。

**Q: 页面加载速度慢？**
A: 已优化脚本加载策略，核心脚本优先加载，非核心脚本延迟加载，提高页面加载速度。

**Q: 数据丢失怎么办？**
A: 系统使用LocalStorage和IndexedDB存储数据，建议定期导出备份数据。
